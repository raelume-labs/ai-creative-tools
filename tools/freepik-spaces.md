# Freepik Spaces

**Website:** [freepik.com/spaces](https://www.freepik.com/spaces)

## Overview

Freepik launched Spaces in November 2025, adding a node-based AI canvas to its massive creative ecosystem. For teams that live inside Freepik's stock library, it's a natural extension that combines AI generation with access to millions of stock assets.

## Key Differentiator

**Stock library integration** - pull in vectors, photos, and templates directly from Freepik's catalog while building AI workflows.

## Core Features

| Feature | Description |
|---------|-------------|
| Canvas Type | Node-based, infinite |
| AI Models | 36+ image models, 9+ video models, audio + editing tools |
| Media Types | Image, Video, Audio |
| Real-time Collaboration | Yes (Figma-style cursors) |
| Stock Library | Massive (Freepik ecosystem) |

## Image Models (36+)

Includes: Flux (with Kontext for text-in-image), Mystic (Freepik's own model built on Flux), Google Imagen, Ideogram, Seedream, Nano Banana Pro, Magnific for upscaling, Runway, GPT/DALL-E, Z-Image Turbo.

## Video Models (9+)

Includes: Google Veo 3, Kling 2.1/2.6/3.0, Runway Gen 4, Seedance, Wan AI, PixVerse 4.5, MiniMax Hailuo 02, Sora.

## Audio

- ElevenLabs
- Sound Effects generation
- Lip Sync API

## Editing Tools

- Image upscaler (up to 10K resolution)
- Video upscaler
- Background remover
- Retouch
- Reimagine (image-to-image variations)
- Image enhancer
- Sketch-to-image
- Video editor

## Pricing

| Plan | Price | Credits/Year |
|------|-------|--------------|
| Free | $0 | Limited (up to 3 Spaces) |
| Essential | $5.75/month | 84K |
| Premium | $12/month | 216K |
| Premium+ | $24.50/month | 540K |
| Pro | $158.33/month | 3.6M |

Note: Credit costs vary by model and settings. A single image can cost 50-500 credits; a 9-second HD video uses ~2,600 credits.

## Limitations

- No 3D generation or Gaussian splatting
- Credit costs can be unpredictable
- "Unlimited" generation only at Premium+ tier
- Advanced features still consume credits

## Best For

- Teams already using Freepik for stock assets
- Workflows centered on 2D image and video content
- Users wanting tight stock library integration
- Moderate-volume users on lower tiers
